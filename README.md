## Getting Started

To start using Dev Quiz App:

1. **Clone the repository:**
   ```sh
   git clone https://github.com/cmtri2005/Dev-Quiz-App.git
   cd Dev-Quiz-App
   ```

2. **Install dependencies:**
   ```sh
   flutter pub get
   ```

3. **Run the project:**
   ```sh
   flutter run
   ```
   Select your preferred platform (Android, iOS, web, etc.) during the run.
